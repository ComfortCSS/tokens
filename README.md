# @comfortcss/tokens

[![npm version](https://img.shields.io/npm/v/@comfortcss/tokens.svg?style=flat-square)](https://www.npmjs.com/package/@comfortcss/tokens)

This package provides a comprehensive set of CSS custom properties (variables) for colors, typography, spacing, radii, shadows, breakpoints, and animations.

## Documentation

A complete description of all variables, including value tables and examples, can be found in the [`/docs`](/docs/README.md) folder.

## Installation

```bash
npm install @comfortcss/tokens@1.x.x
```
