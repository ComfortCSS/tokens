# Breakpoints

Variables for media queries (viewport) and container queries (`inline-size` in `ch`).

| Variable | Default Value | Description |
|----------|---------------|-------------|
| `--breakpoint-sm` | `640px` | Small screen (phones) |
| `--breakpoint-md` | `768px` | Medium screen (tablets) |
| `--breakpoint-lg` | `1024px` | Large screen (laptops) |
| `--breakpoint-xl` | `1280px` | Extra-large screen |
| `--breakpoint-2xl` | `1536px` | Wide-format screens |
