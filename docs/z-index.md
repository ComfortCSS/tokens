# z-index

A hierarchical scale for managing element stacking.

| Variable | Default Value | Description |
|------------|-----------------------|----------|
| `--z-0` | `0` | Base layer |
| `--z-10` | `10` | Level 10 |
| `--z-20` | `20` | 20 |
| `--z-30` | `30` | 30 |
| `--z-40` | `40` | 40 |
| `--z-50` | `50` | 50 |
| `--z-auto` | `auto` | Automatic value |
| `--z-dropdown` | `100` | Dropdowns, menus |
| `--z-sticky` | `200` | Sticky elements (`position: sticky`) |
| `--z-fixed` | `300` | Fixed elements |
| `--z-modal-backdrop` | `400` | Modal backdrop/overlay |
| `--z-modal` | `500` | Modals |
| `--z-popover` | `600` | Popovers, tooltips |
| `--z-tooltip` | `700` | Tooltips |
| `--z-toast` | `800` | Toasts (notifications) |
| `--z-loading` | `900` | Top-level loading indicators |
| `--z-max` | `999` | Maximum layer (use with caution) |
