# Border-radius

A scale of radii ranging from no rounding to a full circle.

| Variable | Default value | Description |
|----------|---------------|-------------|
| `--radius-none` | `0` | No rounding |
| `--radius-sm` | `0.125rem` (~2px) | Small radius |
| `--radius-base` | `0.25rem` (~4px) | Base radius |
| `--radius-md` | `0.375rem` (~6px) | Medium |
| `--radius-lg` | `0.5rem` (~8px) | Large |
| `--radius-xl` | `0.75rem` (~12px) | Extra large |
| `--radius-2xl` | `1rem` (~16px) | Double XL |
| `--radius-3xl` | `1.5rem` (~24px) | Triple XL |
| `--radius-full` | `9999px` | Full circle (ellipse) |
